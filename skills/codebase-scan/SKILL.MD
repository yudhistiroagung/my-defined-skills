# Codebase Scan

## Overview
Use this skill to understand the project structure, architecture, dependencies and code convention.

## When to Use
Use this skill when we want to create new features or refactor existing code.

## Instructions
- **Step 1:** read file `ARCHITECTURE.md` from root project folder and read the content carefully
- **Step 2:** Analyze the entry point of the project, such as `index.js` or `App.js` or `main.js`
- **Step 3:** Identify the main components or modules in the project
- **Step 4:** Analyze the dependencies between the components or modules, and how to use them
- **Step 5:** Identify how all depedencies is wired together
